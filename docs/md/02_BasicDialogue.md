# 消息类型

在对话场景里面可以使用的消息类型，以下消息类型均涉及，详情请参考服务端接口=>消息类型章节。

- 文本
- 表情
- 图片
- 图文
- 视频
- 语音
- 文件
- Ark 消息
- Embed 消息
- Markdown 消息
