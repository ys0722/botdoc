#### ✅事件(4)

##### ✅用户添加机器人

l **基本概况**

| 基本     |                                |
| -------- | ------------------------------ |
| 事件类型 | FRIEND_ADD                     |
| 触发场景 | 用户添加机器人‘好友’到消息列表 |
| 权限要求 | 暂无                           |
| 推送方式 | Websocket                      |

l **事件字段**

| **属性**  | **类型** | **说明**   |
| --------- | -------- | ---------- |
| timestamp | int      | 添加时间戳 |
| openid    | string   | 用户openid |
